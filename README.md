## Descripción
Este trabajo tiene como objetivo demostrar el funcionamiento de la virtualización mediante la creación y ejecución de una máquina virtual Ubuntu utilizando Oracle VirtualBox.

Se analiza el concepto de virtualización, sus ventajas y aplicaciones, además de realizar una práctica donde se ejecuta un sistema operativo Linux dentro de Windows.

## Herramientas utilizadas

- Oracle VirtualBox
- Ubuntu Linux
- Windows 11
- Terminal Bash

## Procedimiento realizado

1. Instalación de Oracle VirtualBox.
2. Creación de una máquina virtual Ubuntu.
3. Configuración de memoria RAM y almacenamiento virtual.
4. Habilitación de virtualización (AMD-V / SVM Mode).
5. Inicio del sistema Ubuntu.
6. Ejecución de comandos básicos en Bash.
7. Obtención de evidencias mediante capturas de pantalla.
